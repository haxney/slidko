# First sessions — ordered task list

**Hardware is NOT required for any of these.** Synthetic captures and any .sr
files already on disk suffice. Do not block on device availability; do not
add hardware-touching code paths to the test suite.

1. **Repo scaffolding.** `pyproject.toml` (Python 3.11+, numpy, scipy,
   pytest; keep the dependency list minimal — no LLM SDK until Phase 5),
   package skeleton `src/slidko/{capture,measure,decode,narrate,corpus}/`,
   pytest wiring, ruff/format config, CI stub that runs tests headless.

2. **.sr file reader** (`capture/srfile.py`). The sigrok session format is a
   zip archive: a `metadata` file (configparser/INI format — samplerate,
   channel list, unitsize, capturefile base name) plus binary chunk files
   (`logic-1-1`, `logic-1-2`, …) of packed sample words. Deliver:
   `read_sr(path) -> Capture` where `Capture` holds per-channel numpy bool
   arrays + samplerate + channel names + provenance metadata. Verify the
   chunk-naming and unitsize handling against a real file generated by
   sigrok-cli before trusting assumptions (flag: format details here are
   MODERATE confidence until checked against an actual capture).

3. **Edge extraction** (`measure/edges.py`). Per-channel rising/falling edge
   timestamps from bit arrays. Vectorized numpy (`np.diff`/`np.flatnonzero`);
   output arrays of (sample_index, polarity). Include interval computation
   helpers. Property tests: synthetic square waves round-trip exactly.

4. **Synthetic capture generator** (`tests/synth.py` — test infrastructure,
   built early because everything downstream consumes it). Generate known
   edge streams in-memory with ground-truth labels: UART frames at arbitrary
   baud (plus SBUS 100000-8E2), I²C transactions (start/stop/ACK/NAK
   structure), SPI bursts (all CPOL/CPHA), WS2812 bit trains with spec-exact
   and deliberately violated timing, PWM/servo, DShot. Parameterize jitter
   and glitch injection — the smoke detector's tests need dirty synthetics.

5. **Interval statistics** (`measure/intervals.py`). Inter-edge interval
   histograms, autocorrelation-based period estimation, dominant-period
   extraction with confidence. These primitives feed every discriminator.

6. **UART auto-baud** (`measure/uart.py`) — the first real discriminator:
   min/GCD of inter-edge intervals snapped to the standard baud table (+SBUS
   exception), idle-level detection, start-bit framing sanity check, numeric
   confidence output. Acceptance criteria in ROADMAP.md Phase 1.

7. **sigrok-cli subprocess wrapper** (`capture/sigrokcli.py`). Timed capture
   from an fx2lafw-class device to .sr; enumerate devices; surface driver
   errors cleanly. Keep it thin; it is a leaf dependency, mockable in tests.
   (Do not attempt in-process libsigrok bindings — subprocess + .sr files is
   the robust integration path.)

After task 7, proceed to the Phase 1 discriminator tree per ROADMAP.md,
building each protocol discriminator against the synthetic generator first
and corpus entries second.
